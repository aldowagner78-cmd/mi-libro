# Imágenes

Esta carpeta contiene las portadas e imágenes del libro.

## Estructura recomendada:

- `portada.jpg` - Portada principal del libro
- `portada-capitulo-01.jpg` - Portada del capítulo 1
- `portada-capitulo-02.jpg` - Portada del capítulo 2
- etc.

## Especificaciones:

- Formato: JPG o PNG
- Resolución recomendada: 1200x1800px para portadas
- Peso máximo: 500KB por imagen
